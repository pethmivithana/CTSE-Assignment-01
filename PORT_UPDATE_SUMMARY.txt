================================================================================
TROUBLESHOOTING
================================================================================

Q: "Cannot connect to User Service" when registering?
A: Check that:
   - User Service is running on port 5002 (not 3001)
   - API Gateway is pointing to http://localhost:5002
   - docker-compose.yml has USER_SERVICE_URL=http://user-service:5002

Q: Frontend cannot reach API Gateway?
A: Check that:
   - API Gateway is running on port 5001
   - Frontend .env has REACT_APP_API_URL=http://localhost:5001
   - docker-compose.yml has REACT_APP_API_URL=http://localhost:5001

Q: Services can't find each other in Docker?
A: Make sure:
   - Using Docker container names (not localhost)
   - docker-compose.yml has correct service DNS names
   - All services are on the same network (feedo-network)

Q: Port already in use?
A: Kill the process:
   lsof -i :5001    # Check what's using port 5001
   kill -9 <PID>    # Kill it
